# LastExecuteRecord Project

This project, named "LastExecuteRecord", is designed to record the last execution time of applications using NTFS data streams. It is a C++ project configured to use the Unicode character set, demonstrating modern practices in application development for Windows environments.

The primary goal of this project is to provide a simple yet effective way of tracking application usage over time, which can be particularly useful for system administration, auditing, and user behavior analysis.

## Features

- Records the last execution time of applications.
- Utilizes NTFS data streams for minimal impact on system performance.
- Developed as a Unicode C++ application for wide character support.
